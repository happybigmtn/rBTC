# Skill Packaging + Clawhub

## Summary
Package the verification, build, update, and mining workflows as an Agent Skill for Clawhub distribution.

## Acceptance Criteria
1. `skill/SKILL.md` documents verify/build/run/mine/update workflows.
2. Skill bundle includes scripts and references with minimal dependencies.
3. Verification outputs are machine-readable and publishable for agent audit.
4. A Clawhub-ready metadata file exists if required by Clawhub.

## Out of Scope
- Clawhub account or publishing workflow beyond local packaging.
